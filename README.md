# Python Loss Reserving Engine

Automated actuarial loss reserving using the Chain Ladder method in Python.

## Problem
Manual Excel-based reserving is time-consuming and difficult to reproduce.

## Method
Classic Chain Ladder using development factors from cumulative claims triangles.

## Features
- Claims triangle ingestion  
- Development factor calculation  
- Ultimate loss estimation  
- IBNR calculation  
- Simple visual diagnostics  

## Tools
Pandas  
NumPy  
Matplotlib
